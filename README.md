# Sistema de Análise de Crédito

Sistema desenvolvido em Java que analisa a concessão de crédito 
com base no salário e endividamento do cliente.

## Funcionalidades

- Verifica se o cliente possui emprego ativo
- Calcula o limite de crédito com base em 30% do salário
- Analisa a dívida atual e aprova ou reprova a solicitação
- Exibe o limite disponível em caso de aprovação

## Regras de negócio

- Cliente deve estar empregado
- Dívida total não pode ultrapassar 30% do salário
- Limite disponível = 30% do salário - dívida atual

## Tecnologias

- Java
- IntelliJ IDEA

## Como executar

1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe `Main.java`
